# Custom font on Kindle
http://2012-2013.chriskrycho.com/web/custom-fonts-on-kindle/

- Create folder "fonts"
- Put 4 ttf files there:
  - FONT_NAME-[Regular,Bold,Italic,BoldItalic]
  - Need to have all fours
  - e.g. Arial-Regular.ttf, Arial-Bold.ttf ...
- Create file "USE_ALT_FONTS" at root of Kindle
